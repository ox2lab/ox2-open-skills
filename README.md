# ox2-open-skills

옥스투랩 AI Skills

# 스킬 설치

## 1. 다운로드 스크립트

```bash
curl -O https://raw.githubusercontent.com/ox2lab/ox2-open-skills/main/install-skills.sh
```

## 2. 스크립트 실행

```bash
sh install-skills.sh [스킬 이름] [스킬 이름] ...
or
sh install-skills.sh [스킬 이름] [스킬 이름] ...
```

`.agents/skills/` 폴더 아래에 설치됩니다.
